# C Programming Internship Projects

This repository contains the projects I developed during my C Programming internship at CODE-A-NOVA.
These projects helped me understand how to apply C concepts to real-world problems.

---

## Projects

### 1. Contact Management System

This project is used to store and manage contacts using file handling.

Features:

* Add new contacts
* View all contacts
* Search contacts by name
* Edit and delete contacts

I used structures and file handling to make sure the data is saved even after the program is closed.

---

### 2. Banking Management System

This is a simple banking system where users can perform basic operations.

Features:

* Create account
* Deposit money
* Withdraw money
* Check balance

In this project, I worked with binary files and learned how to update records using file pointers.

---

### 3. Quiz Game

This is a command-line quiz program with multiple-choice questions.

Features:

* 5 questions with options
* Score calculation
* Final result display
* Score saved in a file

This helped me practice arrays, loops, and conditions.

---

### 4. Unit Converter

This program converts different units like temperature, distance, and weight.

Features:

* Celsius ↔ Fahrenheit
* Kilometer ↔ Miles
* Kilogram ↔ Pounds

It is a simple menu-driven program using functions.

---

## What I Learned

* Basics of C programming
* Working with structures
* File handling (text and binary files)
* Writing menu-driven programs
* Improving logical thinking

---

## How to Run

1. Compile the program:

```
gcc filename.c -o output
```

2. Run the program:

```
./output
```
