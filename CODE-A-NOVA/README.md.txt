# Banking System Mini Project in C

This project was developed as part of the CODE-A-NOVA C Programming Internship.

## Project Description
A simple bank management system implemented using C language.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Data stored permanently using file handling

## Concepts Used
- Structures
- Functions
- Switch Case
- File Handling (Binary files)

## How to Run
1. Open banking.c in Dev C++
2. Compile the program
3. Run the executable file
